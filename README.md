# CRM Premium Dashboard

Modern CRM dashboard built with **HTML, CSS and JavaScript** focused on client management, sales tracking and responsive analytics.

## Live Preview

Designed as a portfolio-ready project for junior developer and freelance opportunities.

## Features

- Add, edit and delete clients
- Search clients instantly
- Register sales by client
- Sales history section
- Dashboard metrics
- Monthly sales chart
- LocalStorage data persistence
- Fully responsive design
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js

## Project Goals

This project was created to demonstrate frontend development skills, UI structure, responsive layouts and JavaScript logic using real business scenarios.

## Future Improvements

- Export data to Excel / PDF
- Dark / Light mode
- Authentication system
- Backend integration
- Real database connection
